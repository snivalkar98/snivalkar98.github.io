# Urja Communication — Static Tech Portfolio (No Backend)

This is a **static** (non-dynamic) portfolio with a modern, tech-savvy UI.
You can edit all content by changing `data/content.json` (recommended) or directly in `index.html`.

## Quick edit
- Update your name, role, summary, skills: `data/content.json`
- Update projects: `data/content.json` → `projects[]`
- Update experience: `data/content.json` → `experience[]`
- Replace images in: `assets/img/` and `assets/img/projects/`

## Run locally
Open `index.html` directly OR use a simple server:
```bash
python3 -m http.server 8080
```
Then visit: http://localhost:8080

## Deploy
- GitHub Pages / Netlify / any static hosting.
